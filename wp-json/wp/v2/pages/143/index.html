CPS-IoTBench – Workshop on Benchmarking Cyber-Physical Networks and Internet of Things<\/h2>\n

 <\/p>\n

The prime goal of CPS-IoTBench\u00a0is to be an exchange platform for academics and industrials\u00a0dealing with all facets\u00a0of rigorously evaluating and comparing cyber-physical networks and systems. The workshop is meant to be a\u00a0venue for learning about each other\u2019s challenges and evaluation methodologies and for debating future research agendas to jointly define the performance metrics and benchmarking scenarios that matter from an overall system\u2019s perspective.<\/p>\n

EWSN’s\u00a0Dependability Competition<\/h2>\n

This competition aims to compare the performance of different WSN\/IoT communication protocols in environments rich with radio interference.<\/p>\n

Each year, the competition’s scenario is added to the IoTBench problem set.<\/p>\n
